# skill-cartridge-registry

![Cocapn Vessel](https://img.shields.io/badge/cocapn-vessel-purple)

JSON skill cartridge marketplace — discover, rate, install cartridges for any vessel

Part of the [Cocapn fleet](https://github.com/Lucineer/the-fleet).

## Endpoints
| Path | Purpose |
|------|---------|
| / | Landing page |
| /api/cartridges | Browse cartridges |
| /api/rate | Rate a cartridge |
| /health | Liveness check |

## Deploy
Fork, deploy with wrangler.
